# Cherry Burn 

This is the repository of Cherry Burn a simple Finance Mangement Web Application built with using WAMP stack (Windows - Apache - MySQL - PHP)

## Project Structure

```
    /project-root
    ├── public/ # Web root - all browser-accessible files
    │ ├── index.php
    │ ├── login.php
    │ ├── dashboard.php
    │ ├── logout.php
    │ └── assets/
    │ ├── css/
    │ ├── js/
    │ └── images/
    ├── includes/ # Backend PHP logic and database configs
    │ ├── config.php
    │ └── auth.php
    ├── sql/ # SQL scripts
    │ └── create_users.sql
    └── README.md
```

## Tech Stack

- PHP 8.x
- MySQL / MariaDB
- Apache (via XAMPP)
- HTML/CSS/JS (Vanilla)

## Setup 

First install [XAMPP](https://www.apachefriends.org/) is you haven't already. 

Start an Apache server and a MySQL server

The application should be available [here](http://localhost/)
